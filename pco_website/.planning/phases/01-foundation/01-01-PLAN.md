**Replace `app/layout.tsx` entirely** (remove Geist imports, add EB Garamond + Cormorant Garamond):

```typescript
import type { Metadata } from "next";
import { EB_Garamond, Cormorant_Garamond } from "next/font/google";
import { Toaster } from "sonner";
import "./globals.css";

const ebGaramond = EB_Garamond({
  subsets: ["latin"],
  variable: "--font-eb-garamond",
  display: "swap",
});

const cormorantGaramond = Cormorant_Garamond({
  subsets: ["latin"],
  weight: ["300", "400", "500", "600", "700"],
  variable: "--font-cormorant-garamond",
  display: "swap",
});

export const metadata: Metadata = {
  title: "Psi Chi Omega",
  description: "Psi Chi Omega Fraternity",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html
      lang="en"
      className={`${ebGaramond.variable} ${cormorantGaramond.variable}`}
    >
      <body>
        {children}
        <Toaster theme="dark" richColors position="top-right" />
      </body>
    </html>
  );
}
```

Critical details:
- `@theme inline` (not plain `@theme`) — required when value is `var(--something)`. Without `inline`, font utilities fail silently.
- Cormorant Garamond requires `weight: ["300", "400", "500", "600", "700"]` explicitly — it is NOT a variable font.
- `<Toaster>` is safe in a Server Component (root layout) — `toast()` calls come from Client Components in later plans.
- Remove ALL existing Geist font imports and related code from layout.tsx.
