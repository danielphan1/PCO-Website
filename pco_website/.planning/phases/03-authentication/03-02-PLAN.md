**1. Update `contexts/AuthContext.tsx` — replace the useEffect stub:**

Replace the entire `useEffect` block (lines ~26-38) with:

```typescript
useEffect(() => {
  const token = getAccessToken();
  if (!token) {
    setLoading(false);
    return;
  }
  apiFetch<User>("/v1/users/me")
    .then((data) => setUser(data))
    .catch(() => {
      // 401 with exhausted refresh, or network error — clear stale tokens
      clearTokens();
      document.cookie = "auth-hint=; path=/; max-age=0; samesite=lax";
      setUser(null);
    })
    .finally(() => setLoading(false));
}, []);
```

Add `apiFetch` import at the top: `import { apiFetch } from "@/lib/api";`

CRITICAL: `loading=false` is set ONLY in `.finally()`. Never in the try or happy path. If `loading=false` fires before the fetch resolves, AuthGuard sees `user=null` while `loading=false` and redirects to /login even for authenticated users.

Do NOT modify the `login()` or `logout()` functions — they are already correct.

**2. Create `app/(member)/dashboard/page.tsx` — minimal stub for AUTH-06 verification:**

This stub exists so the access-denied redirect has a destination. Phase 4 will expand it into the full dashboard.

```tsx
"use client";

import { useEffect } from "react";
import { useRouter, useSearchParams } from "next/navigation";
import { toast } from "sonner";

export default function DashboardPage() {
  const router = useRouter();
  const searchParams = useSearchParams();

  useEffect(() => {
    if (searchParams.get("access_denied") === "1") {
      toast.error("Access denied. Admin privileges required.");
      router.replace("/dashboard");
    }
  }, [searchParams, router]);

  return (
    <div className="min-h-screen bg-black flex items-center justify-center">
      <p className="font-body text-white/40 tracking-widest uppercase text-sm">
        Dashboard — coming in Phase 4
      </p>
    </div>
  );
}
```

This page is protected by `app/(member)/layout.tsx` which wraps it in `AuthGuard requiredRole="member"` — unauthenticated users are redirected to /login before they ever reach this component.
